# AI Web Lab (Tank, Word, Philosopher)

이 저장소는 브라우저에서 바로 실행되는 세 가지 웹앱을 포함합니다.

- `tank/` : 간단한 탱크 게임 (키보드/터치 조작)
- `word/` : 단어 외우기 퀴즈 (오답노트 저장)
- `philo/`: 매일 철학 질문 앱 (내 답변 기록)

## 빠른 시작
각 폴더의 `index.html`을 브라우저로 열면 실행됩니다.

## GitHub Pages 배포 (한 저장소에 폴더 3개)
1. GitHub에 `ai-web-lab` 저장소를 만듭니다.
2. 코드 업로드 후, **Settings → Pages**에서
   - Source: `Deploy from a branch`
   - Branch: `main` / `/root` (또는 `/docs`) 선택 후 Save
3. 배포가 끝나면 아래처럼 접근합니다.
   - `https://<github-username>.github.io/ai-web-lab/tank/`
   - `https://<github-username>.github.io/ai-web-lab/word/`
   - `https://<github-username>.github.io/ai-web-lab/philo/`
